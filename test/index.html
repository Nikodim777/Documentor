    Функции модуля

    Назад

    aprint

    Функция предназначена для форматированного вывода одномерного или двумерного массива.

    Аргументы:

    titleУказатель на строку заголовка печатаемого массива.
    printУказатель на функцию печатающую элемент определённого типа.
    Назад

    Appropriate

    Функция принимает в качестве параметров указатель на элемент массива и возвращает RES_NOT в случае несоответствия и RES_OK в случае соответствия.

    Аргументы:

    typeТип соответствия.
    checkElУказатель на функцию проверки соответствия элемента массива определённому условию.

    Результат: Возвращает или признак соответствия/несоответствия или число соответствующих элементов в зависимости от типа соответствия

    Группы функций модуля

    Назад

    printFuncs

    Функции служат для вывода значений базовых типов.
    Можно создать аналогичные собзственные функции для вывода значений любого типа.
    Пример_функции_вывода_элемента
    void printDbl(const void *arg)
    {
    printf("%.6f\t", *(double*)arg);
    }

    Аргументы:

    argУказатель на значение которое нужно вывести.

    Список функций:

    void printInt(const void *arg)
    void printFlt(const void *arg)
    void printDbl(const void *arg)
    void printExp(const void *arg)
    void printCmplx(const void *arg)
    Назад

    fillFuncs

    Функции служат для заполнения массива обределённым образом.
    step - заполняет целыми числами по порядку.
    stepTwc - то же но с двойным шагом.
    Функции с суффиксом Rnd - заполняют случайными числами. Можно создать аналогичные собственные функции. Пример_функции_изменения_элемента void setRnd(void* masEl, largeSize_tp num)
    {
    *(int*)masEl = rand() % 100;
    }

    Аргументы:

    masElУказатель на элемент массива значение которого нужно изменить.
    numПорядковый номер элемента.

    Результат: Ничего не возвращает

    Список функций:

    void step(void* masEl, largeSize_tp num)
    void stepTwc(void* masEl, largeSize_tp num)
    void setRnd(void* masEl, largeSize_tp num)
    void setDblRnd(void* masEl, largeSize_tp num)

    Назад

    checkRes

    Перечисление хранящее результат соответствия элементов массива некоторому условию

    Константы перечисления:

    RES_OK Соответствует
    RES_NOT Не соответствует
    Назад

    typeAppr

    Перечисление хранящее тип соответствия элементов массива некоторому условию

    Константы перечисления:

    ALL_APPR Все элементы соответствуют
    ANY_APPR Хотя бы один элемент соответствуют
    CNT_APPR Определённое число элементов соответствуют

    printMacro

    Макросы для печати массива определённого типа

    Список макросов:

    aprinti(mas,size1,size2,title)aprint((mas),sizeof(int),(size1),(size2),(title),printInt);

    aprintd(mas,size1,size2,title)aprint((mas),sizeof(double),(size1),(size2),(title),printDbl);
    aprinte(mas,size1,size2,title)aprint((mas),sizeof(double),(size1),(size2),(title),printExp);
    aprintc(mas,size1,size2,title)aprint((mas),sizeof(double complex),(size1),(size2),(title),printCmplx);
